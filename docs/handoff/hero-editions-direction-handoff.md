# Hero Editions Direction Handoff

## 기준 레퍼런스

- 목표 무드는 Shopify Editions Winter 2026:
  - https://www.shopify.com/editions/winter2026
- 이번 세션에서는 위 사이트를 실제로 열어 현재 로컬 UI와 비교했다.

## 현재 UI를 보고 내린 결론

- 현재 히어로는 이전보다 훨씬 낫다.
- 특히 마지막 컷에서 고양이가 왼쪽 아래에 있고, 오른쪽 공중에 프로젝트 전광판이 뜨는 구성은 방향 자체는 맞다.
- 하지만 전체 흐름은 아직 Shopify Editions처럼 "짧은 스크롤로 큰 변화가 이어지는" 느낌보다는
  "조금 오래 내려서 마지막 장면에 도착하는" 쪽에 가깝다.

## 실제로 보인 문제

### 1. 스크롤 길이가 과하다

- 현재 히어로는 `560vh`라서 사용자가 마지막 전광판까지 도달하는 데 체감상 오래 걸린다.
- 장면 전환 품질과 별개로, 도착 보상이 나오기까지 스크롤 부담이 먼저 느껴진다.
- Shopify Editions 계열은 길더라도 스크롤 한 번마다 상태 변화가 강하게 읽히는데, 현재 포트폴리오는 그 체감이 상대적으로 약하다.

### 2. 컷 수가 많아 흐름이 늘어진다

- 지금 구조는 `6컷 + 전광판 + 별도 Projects 섹션`이다.
- 장면이 많다기보다, "중간 컷을 지나 최종 도착점에 간다"는 리듬이 길다.
- 다음 세션에서는 6컷을 유지하는 것보다 `3~4컷`으로 압축하는 쪽이 더 유리하다.

### 3. 사이드바가 현재 무드와 안 맞는다

- 현재 오른쪽 고정 사이드바는 기능적으로는 문제 없지만, 시네마틱 히어로 위에서는 "편집형 내비게이션"보다 "앱 위젯"처럼 보인다.
- Shopify Editions와 비교했을 때 가장 먼저 이질감이 느껴지는 요소 중 하나다.
- 지금 히어로 단계에서는 제거하거나, 최소한 히어로 구간에서는 숨기는 편이 더 낫다.

### 4. 전광판 위치가 조금 낮다

- 현재 전광판은 고양이가 보는 방향은 맞지만, 세로 위치가 살짝 낮다.
- 조금 더 위로 올리면 "고양이가 공중 전광판을 올려다본다"는 읽힘이 더 강해질 수 있다.

### 5. Projects 섹션이 전광판의 임팩트를 분산시킨다

- 현재 아래 `Projects` 섹션은 정보 구조상 나쁘지는 않다.
- 하지만 지금 연출 목표에서는 전광판이 이미 클라이맥스 역할을 하고 있어서, 바로 아래 별도 프로젝트 섹션이 나오면 도착 감각이 분산된다.
- 당분간은 `Projects` 섹션을 제거하는 편이 맞다.

## 다음 세션 권장 방향

### 1순위

- 히어로 길이를 줄인다.
- 목표 범위는 대략 `280vh ~ 360vh`.
- 현재처럼 긴 체류형보다는 "짧게 밀면 장면이 강하게 바뀌는" 리듬으로 재구성한다.

### 2순위

- 컷 수를 `3~4컷`으로 줄인다.
- 추천 구조:
  - 도입 컷
  - 이동/전개 컷
  - handoff 컷
  - 전광판 등장 컷

### 3순위

- 전광판을 지금보다 살짝 위로 올린다.
- 오른쪽 공중에 떠 있는 느낌은 유지하되, 고양이 시선선보다 조금 더 높은 곳에 배치한다.

### 4순위

- `Projects` 섹션은 일단 제거한다.
- 전광판이 현재 히어로의 도착점 역할을 전담하도록 만든다.

### 5순위

- 전광판 클릭 시 모달 또는 전체 오버레이 상세 뷰를 연다.
- 즉, 아래 섹션으로 내려가는 대신 히어로 안에서 상세 경험을 이어가는 방식이 더 적합하다.
- 현재 사용자 의도상 "전광판을 누르면 프로젝트가 열린다"가 더 자연스럽다.

### 6순위

- 사이드바는 제거 또는 축소한다.
- 최소한 히어로 구간에서는 숨기고, 이후 정보 섹션이 남아 있을 때만 다시 나타나는 구조도 가능하다.

## 유지할 것

- 도시 + 고양이 무드 자체는 유지
- 마지막 컷 위 공중 전광판이라는 구조 유지
- 차분한 블루 계열과 레이어감 유지
- 프로젝트를 "카드 목록"보다 "전시 오브젝트"로 보여주는 방향 유지

## 한 줄 요약

- 현재 버전은 방향은 맞지만 아직 길다.
- 다음 세션은 "더 많이 만들기"보다
  `스크롤 압축 + 사이드바 제거 + 전광판 상향 + Projects 섹션 제거 + 클릭 모달`
  순서로 단순화하는 것이 맞다.
