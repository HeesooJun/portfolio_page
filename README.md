# Heesoo Jun Portfolio

포트 폴리오 제작중

## 🛠️ 사용 기술
- Next.js 15 (App Router)
- React 19 + TypeScript 5
- Tailwind CSS 4
- next/font (Google Inter)

## 📂 프로젝트 구조
```
├─ public/ # 사진
├─ src/
│  └─ app/
│     ├─ layout.tsx  # 전역 폰트 및 배경
│     ├─ page.tsx    # 섹션 조합
│     └─ section/    # Hero, Projects, About, Tech, Footer 컴포넌트
```

## 🚀 실행 방법
```bash
npm install
npm run dev
```
